**네트워크 게임플레이 텀 프로젝트**
 - 팀원 소개
   - 김동재 : 팀장 및 서버 개발
   - 서동우 : 서버 개발 및 클라이언트 개발
   - 임윤수 : 문서 작업 및 보조 개발
 - 개발 기간 : 2023.10 ~ 2023.11
 - 장르 : 2인 슈팅 대결
 - 기술 스택 : C++, DirectX 12, WinSock2

**프로젝트 관리**
 - 프로젝트 기간동안 일일 단위 일정 관리
 - 매주 1회 팀원 회의를 통해 진행 상황 공유
 - 문제 상황 로그화를 통한 유연한 일정 관리

**기술적 도전**
- WinSock2의 블럭킹 API를 사용해 멀티플레이 게임 개발
- 블럭킹/동기 API를 활용하기 위해 스레드 동기화 구조를 철저하게 설계
